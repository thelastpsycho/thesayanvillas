DESIGN GUIDELINES
	•	Framework: Tailwind CSS
	•	Color Palette: Soft whites, beige, muted gold, sand, charcoal accents.
	•	Layout: Spacious, clean, minimalistic, centered design with large visuals.
	•	Typography: Playfair Display for headlines, Inter or DM Sans for body text.
	•	Navigation: Sticky and minimal with smooth scrolling and active state indicator.
	•	Hero Image: Full-width rice field or villa pool with light overlay text.
	•	Animations: Soft fade-ins, gentle transitions, elegant hover effects.
	•	Responsiveness: Fully responsive and mobile-first.
	•	Performance: Optimized for fast loading, clean HTML structure, SEO-friendly.
